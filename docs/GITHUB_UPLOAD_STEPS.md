# GitHub Upload Steps

1. Create a new GitHub repository.
2. Suggested repository name:

```text
led-lettuce-photosynthesis-fluorescence
```

3. Keep the repository public only if the dataset, figures, and source labels can be shared publicly.
4. Do not initialize the repository with a README, because this project already has one.
5. Upload the full contents of this folder:

```text
README.md
data/
figures/
scripts/
docs/
```

6. Commit the files with a neutral message, for example:

```text
Initial reproducible LED-spectrum analysis
```

7. Check that the README displays all four figures correctly.
8. Before sharing the repository, do a final search for private notes, draft comments, or application-specific wording.
